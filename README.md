# OpenMP-Sorting
Parallel sorting algorithm with OpenMP.

## Compilation

Dependencies:
- OpenMP (libgomp)

Build the program:

    make [all]

Clean-up:

    make distclean

Help:

    make help

## Usage

    ./ompsort

## Report

Dependencies:
- TexLive Full
  
Build the report:

    make report

Visualization:

    [PDF VIEWER] ./report/report.pdf
